# 🚀 1단계 - 지뢰 찾기(그리기)

## 기능 요구사항

- 높이와 너비, 지뢰 개수를 입력받을 수 있다.
- 지뢰는 눈에 잘 띄는 것으로 표기한다.
- 지뢰는 가급적 랜덤에 가깝게 배치한다.

### 기능 리스트

- [x] 높이를 입력받는다.
- [x] 넓이를 입력받는다.
- [x] 지뢰 개수를 입력받는다.
- [x] `지뢰개수 > h x w` 이면 exception throw
- [x] `Board`의 각 칸(`Cell`)은 지뢰칸 또는 빈칸이다. 
- [x] 지뢰는 랜덤에 배치하고 겹치면 안된다.
- [x] 그린 게임 판을 콘솔에 출력한다.
